# node-red-artik
Integrate [NODE-RED](https://nodered.org) with [Samsung Artik](https://www.artik.io/) for [MIMIC MQTT Simulator](http://gambitcomm.com/site/mqttsimulator.php)

Flows to add/delete devices in bulk to Samsung Artik.
